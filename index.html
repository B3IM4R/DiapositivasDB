¿QUÉ ES UNA BASE DE DATOS?

Es la encargada de proporcionar una forma estructurada y organizada de almacenar datos, lo que permite un acceso rápido y eficiente a la información

¿QUÉ ES SQL?

(Lenguaje de Consulta Estructurada)

Es un lenguaje de programación que utilizan casi todas las bases de datos relacionales para consultar, manipular y definir los datos, además de para proporcionar control de acceso

Tipos de Bases de Datos

  • Relacionales
  • En Columnas
  • Distribuidas
  • NoSQL
  • En la Nube
  • Documentales

¿Qué Base de Datos Usar?

Para decidir, debes reflexionar en la relación entre los diferentes tipos de datos que vas a almacenar y de acuerdo a eso debes elegir la mejor te convenga
